# Blinkit Grocery Sales Analysis using Power BI

This project presents an end-to-end Business Intelligence and Data Analytics solution for analyzing Blinkit's grocery sales performance, customer satisfaction, inventory distribution, and outlet performance using Power BI.

---

# Project Overview

The main objective of this project is to transform raw grocery sales data into actionable business insights through interactive dashboards and KPI analysis.

The dashboard helps analyze:

* Sales performance
* Customer behavior
* Product category trends
* Outlet performance
* Inventory distribution
* Customer satisfaction metrics

---

# Business Requirements

The project focuses on identifying key insights and optimization opportunities using various KPIs and visualizations.

## KPI Requirements

* Total Sales
* Average Sales
* Number of Items Sold
* Average Customer Rating

---

# Key Analysis Performed

## 1. Total Sales by Fat Content

* Analyzed how fat content impacts overall sales
* Compared KPIs across different fat categories

## 2. Total Sales by Item Type

* Identified best-performing product categories
* Compared sales distribution across item types

## 3. Fat Content by Outlet

* Compared outlet sales segmented by fat content

## 4. Total Sales by Outlet Establishment

* Evaluated outlet performance based on establishment type and age

## 5. Sales by Outlet Size

* Analyzed relationship between outlet size and sales percentage

## 6. Sales by Outlet Location

* Assessed geographic sales distribution

## 7. All Metrics by Outlet Type

* Comprehensive KPI comparison across outlet types
